# AI Code Helper

一个基于 Spring Boot 和 LangChain4j 的智能 AI 代码助手项目，集成通义千问大模型，提供流式对话、RAG（检索增强生成）、MCP（模型上下文协议）等功能。


## 🎯 项目简介

AI Code Helper 是一个智能化的代码辅助工具，旨在帮助开发者提高编程效率和学习效果。项目结合了先进的 AI 技术和丰富的学习资源，提供代码建议、学习路线规划、面试技巧指导等功能。

## 🛠️ 技术栈

### 后端
- **Java 21** - 编程语言
- **Spring Boot 3.4.1** - Web 框架
- **LangChain4j 1.1.0** - LLM 应用开发框架
- **通义千问 (Qwen-max)** - 大语言模型
- **Lombok** - 简化 Java 代码
- **Maven** - 项目管理工具

### 前端
- **Vue 3.4.0** - 渐进式 JavaScript 框架
- **TypeScript 5.3.0** - 类型安全的 JavaScript 超集
- **Vite 5.0.0** - 现代前端构建工具
- **Axios 1.6.0** - HTTP 客户端

## ✨ 功能特性

- ✅ **智能对话** - 基于通义千问大模型的流式对话体验
- ✅ **记忆管理** - 支持多会话记忆，保持上下文连贯
- ✅ **RAG 检索增强** - 集成知识库，提供更准确的答案
- ✅ **MCP 支持** - 模型上下文协议集成
- ✅ **安全守卫** - 输入内容安全检查
- ✅ **工具扩展** - 面试问题工具等实用功能
- ✅ **学习资源** - 内置学习路线、面试技巧等文档
- ✅ **CORS 配置** - 跨域请求支持
