文档四：高频面试题 + 面试技巧
一、高频基础面试题
变量、数据类型、作用域
面向对象：封装、继承、多态
HTTP/HTTPS 区别、状态码
数据库索引、事务、SQL 优化
线程、进程、异步、并发
常见算法：两数之和、反转链表、二叉树遍历
项目难点、如何解决、如何优化
二、前端高频题
原型与原型链
闭包、作用域、this
虚拟 DOM、diff 算法
跨域解决方案
防抖节流、性能优化
三、后端高频题
Spring Bean 生命周期
MyBatis 缓存机制
Redis 应用场景、持久化
消息队列使用场景
高并发解决方案
四、通用面试技巧
自我介绍控制在 1 分钟
不会就说思路，不要瞎编
项目一定要熟悉，能讲清楚架构
反问环节准备问题：团队规模、技术栈、成长路径
态度真诚、逻辑清晰、语速适中
面试后复盘错题，查漏补缺
