文档二：清晰编程学习路线（通用版）
一、入门阶段（1–2 个月）
选一门入门语言：Python / Java / JavaScript 三选一
掌握基础语法：变量、数据类型、条件、循环、函数
学会调试：看懂报错、断点调试、打印日志
做小练习：计算器、猜数字游戏、简单爬虫脚本
二、核心基础（2–3 个月）
数据结构：数组、链表、栈、队列、哈希表、树
算法基础：排序、查找、递归、贪心、简单动态规划
计算机基础：操作系统、网络、数据库基本概念
版本控制：Git 常用命令、GitHub 使用
三、方向进阶（3–6 个月）
前端：HTML/CSS/JS → 框架（Vue/React）→ 工程化
后端：数据库 → 框架（SpringBoot/Django/Express）→ API 开发
数据分析 / AI：Python 数据分析库 → 机器学习基础
移动端：Android / iOS / Flutter / UniApp
四、实战与就业（持续）
做完整项目
刷算法题
写技术博客
参与开源或团队项目
